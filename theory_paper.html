Liquidity Potential Model (LPM) v2

This HTML version renders formulas via MathJax. The Markdown version uses $ and $$ for GitHub rendering.

1) Mechanism recap

Let $p_t$ be price (not log price). Let $\mathcal{L}_t = \{\ell_{1,t}, \dots, \ell_{N,t}\}$ be the set of levels visible at time $t$. Each level has strength $w_{i,t} \ge 0$ and width $\sigma_{i,t} > 0$.

Liquidity potential:

$$ U(p,t) = \sum_{i=1}^{N} w_{i,t}\,\exp\left(-\frac{(p-\ell_{i,t})^2}{2\sigma_{i,t}^2}\right) $$

Drift implication

$$ \mu(p,t) = -\kappa\,\frac{\partial U}{\partial p}(p,t) $$

2) Walk-forward levels (no look-ahead)

In fold $k$, build levels using only history up to $T^{(k)}_{\text{train,end}}$ and apply them to the subsequent test window.

3) Purged walk-forward + embargo

With forward horizon $h$, samples overlap. Purge training samples whose label end overlaps the test period; optionally embargo after the test window.

4) Fractional Differentiation

$$ \tilde p_t = \sum_{k=0}^{K} w_k(d)\,p_{t-k},\quad w_0=1,\; w_k(d) = -w_{k-1}(d)\,\frac{d-k+1}{k} $$

5) Falsifiable hypotheses

  • H1: conditional outcome probabilities around levels improve out-of-sample.
  • H2: repeated touches reduce rejection probability (depletion).
  • H3: multi-timeframe confluence improves reaction odds.
